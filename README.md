# vue-ts-calendar

<img src="https://i.imgur.com/WqTJJNj.png" width="48">

[View Demo](https://vue-ts-calendar.netlify.app/)

A Calendar built with Vue.js using TypeScript and vue-property-decorator

Calendar Types:
* Dual View (Default)
* Single View
* Infinite view

Selection Types:
* Range (Default)
* Single
