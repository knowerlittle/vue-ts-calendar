# vue-ts-calendar

[View Demo](https://vue-ts-calendar.netlify.app/)

A Calendar built with Vue.js using TypeScript and vue-property-decorator

Calendar Options include:
* Dual View (Default)
* Single View
* Infinite view

Selection Otpions include:
* Range (Default)
* Single
